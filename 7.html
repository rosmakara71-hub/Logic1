ល្បែងគណិតវិទ្យា៖ សមីការ

💡 រូបមន្តត្រូវចាំ៖

១. សមីការដឺក្រេទី១: $ax + b = 0$ $(a \neq 0)$

  • បើ $a \neq 0$, សមីការមានឫសតែមួយគត់: $x = -\frac{b}{a}$
  • បើ $a = 0, b = 0$, សមីការមានឫសរាប់មិនអស់ (គ្រប់តម្លៃ $x$ ជាចំនួនពិត)
  • បើ $a = 0, b \neq 0$, សមីការគ្មានឫស

២. សមីការដឺក្រេទី២: $ax^2 + bx + c = 0$ $(a \neq 0)$

  • ឌីស្គ្រីមីណង់ ($\Delta$): $\Delta = b^2 - 4ac$
  • បើ $\Delta > 0$: មានឫសពីរផ្សេងគ្នា $x_1 = \frac{-b - \sqrt{\Delta}}{2a}, x_2 = \frac{-b + \sqrt{\Delta}}{2a}$
  • បើ $\Delta = 0$: មានឫសឌុប $x_1 = x_2 = -\frac{b}{2a}$
  • បើ $\Delta < 0$: គ្មានឫសក្នុងសំណុំចំនួនពិត $\mathbb{R}$ ទេ

៣. ករណីពិសេសនៃសមីការដឺក្រេទី២:

  • បើ $a + b + c = 0$: ឫសគឺ $x_1 = 1, x_2 = \frac{c}{a}$
  • បើ $a - b + c = 0$: ឫសគឺ $x_1 = -1, x_2 = -\frac{c}{a}$

៤. ទ្រឹស្តីបទវីយ៉ែត:

  • ផលបូករឹស: $x_1 + x_2 = -\frac{b}{a}$
  • ផលគុណរឹស: $x_1 \cdot x_2 = \frac{c}{a}$
